Vous ne vous rappelez plus de l'endroit où vous avez enregistré le fichier universe-formula.La seule chose dont vous vous souvenez est que vous l'avez mis quelquepart dans un sous répértoire de /tmp/documents.

Implémentez la méthode String locateUniverseFormula() qui devra trouver le fichier et retourner le chemin complet.
Si le fichier n'existe pas, la méthode doit retourner null.

Exemple : 
retourne /tmp/documents/a/b/c/universe-formula si universe-formula est trouvé dans le répertoire /tmp/documents/a/b/c
